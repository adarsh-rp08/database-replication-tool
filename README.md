# Database Replication & Synchronization Tool

## Overview

A Flask-based Database Replication and Synchronization Tool developed using Python and MySQL. The system enables dynamic database discovery, object browsing, incremental replication, synchronization tracking, audit logging, and role-based access control.

## Features

* User Authentication
* Role-Based Access Control (RBAC)
* Dynamic Database Discovery
* Database Explorer
* Data Preview
* Incremental Database Replication
* Synchronization History
* Audit Logging
* Dashboard Monitoring

## Technologies Used

* Python
* Flask
* MySQL
* HTML
* Bootstrap

## System Modules

### Login Module

Provides secure user authentication.

### Dashboard Module

Displays database count, synchronization jobs, and audit log statistics.

### Replication Module

Allows administrators to replicate data between source and destination databases.

### Database Explorer

Allows users to browse databases, tables, and records.

### Synchronization History

Displays previous synchronization jobs and statistics.

### Audit Logs

Tracks user activities and replication operations.

## Installation

```bash
pip install -r requirements.txt
python app.py
```
