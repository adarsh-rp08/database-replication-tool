# Database Replication & Synchronization Tool

## Overview

A Flask-based database replication tool that performs incremental synchronization between MySQL databases.

## Features

- User Authentication
- Role-Based Access Control (RBAC)
- Dynamic Database Discovery
- Database Explorer
- Data Preview
- Incremental Replication
- Synchronization History
- Audit Logging

## Technologies Used

- Python
- Flask
- MySQL
- HTML
- Bootstrap

## Installation

```bash
pip install -r requirements.txt
python app.py
```
